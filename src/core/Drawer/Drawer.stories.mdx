# Drawer

`Drawer` modal component provides a solid foundation for creating dialogs, popovers,
lightboxes, or whatever else.

## Default variant:

<Canvas>
  <Story name="Default" args={{ variant: "temporary", anchor: "left" }}>
    {(args) => {
      const [open, setOpen] = useState(false);
      return (
        <div>
          <ButtonFilled onClick={() => setOpen(true)}>Open Drawer</ButtonFilled>
          <Drawer
            {...args}
            open={open}
            onClose={() => setOpen(false)}
            modalActions={
              <ButtonOutlined
                style={{ float: "right" }}
                onClick={() => setOpen(false)}
              >
                &#x2715;
              </ButtonOutlined>
            }
          >
            <div
              style={{
                padding: "4rem",
                fontSize: "2rem",
              }}
            >
              This is a Drawer
            </div>
          </Drawer>
        </div>
      );
    }}
  </Story>
</Canvas>
